# 꾸밍 (Kkuming)

꿈 기록과 AI 해몽을 제공하는 앱입니다.

## 주요 기능

- 꿈 기록 및 관리
- AI 기반 꿈 해몽
- 꿈 분석 통계
- 프로필 관리

## 기술 스택

- Flutter
- Supabase
- Firebase
- OpenAI API
- Google AdMob
